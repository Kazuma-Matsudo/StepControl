ブラウザ操作をできるchromeの拡張機能を作成したいです。

- 機能
    - ワークフローとして、1つ1つの処理を設定できる
    - 処理の1つ1つの間に実行間隔を設定できる
    - 操作したいこと
        - クリック
        - フォームの入力
        - コピー機能
            - 文字のコピー
            - URLの文字なら 文字列(URL)でコピー
            - コピー結果としては
                - ラベル設定できる
                    - ラベル設定しなくてもいい
                - ラベル設定できる場合
                    - ```
                        ■ラベル1
                        コピー文字列
                        ■ラベル2
                        コピー文字列
                    ```
                - ラベル設定しない場合
                    - ```
                    コピーした文字列
                    コピーした文字列
                    ```
    - 操作内容のエクスポート、インポート機能が欲しいです(jsonファイルとか？)
    - 処理途中に停止ボタンで動作が止まるようにしたい

● 仕様書とコードベースを踏まえて、現状では対応できない主なパターンを挙げます。                                                                                                                                                                                                                                                                                                                                                                                                
  ---                                                                                                                                                                                                                                    待機・条件系
                                                                                                                                                                                                                                       
  - 要素の出現待ち: 非同期で描画される要素（SPAのルート遷移後、API応答後のDOM）に対して「出るまで待つ」ができない。固定の interval で代用するしかなく、タイミングが不安定
  - 条件分岐: 「要素Aが存在すればクリック、なければスキップ」のようなif/else的な制御がない
  - ループ: 一覧の全行に同じ操作を繰り返す（ページネーション含む）ができない

  ページ遷移・タブ操作系

  - ページ遷移後の継続: クリックで別ページに遷移すると Content Script が消えるため、遷移後のステップが失敗する可能性がある
  - 新規タブ/ウィンドウ操作: target="_blank" で開いたタブへの操作切替ができない
  - iframe 内の操作: iframe 内の要素はメインドキュメントの querySelectorAll では取得できない

  入力操作系

  - キーボード操作: Enter/Tab/Escape/矢印キー等のキー送信ができない（オートコンプリートの確定、モーダルの閉じ等に必要）
  - ファイルアップロード: <input type="file"> への操作は element.value = では設定不可
  - ドラッグ＆ドロップ: ページ上の要素のD&D操作（Kanbanボードの並べ替え等）
  - チェックボックス/ラジオの状態指定: click で切り替えはできるが、「ON にする」「OFF にする」という冪等な操作ができない（現状は単にトグル）

  データ・フロー系

  - 動的な値の参照: コピーで取得した値を後続の input ステップに渡す（変数的な仕組み）ができない
  - 外部データ入力: CSVやスプレッドシートから値を読み込んで順に入力するバッチ処理ができない
  - コピー結果のフォーマット制御: 区切り文字やCSV形式など、出力フォーマットのカスタマイズができない

  実行制御系

  - Service Worker の30秒制限: 長い interval（25秒超）でタイマーが発火しないリスクがある（仕様書の既知の制約に記載済み）
  - スケジュール/定期実行: 決まった時刻やインターバルでワークフローを自動実行する仕組みがない
  - 複数ワークフローの連結: ワークフローAの後にワークフローBを実行する、といった連携ができない
