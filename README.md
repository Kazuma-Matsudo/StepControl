# StepFlow

ブラウザ操作をワークフローとして自動化する Chrome 拡張機能です。
クリック・入力・コピーなどの操作をステップとして組み立て、順番に自動実行できます。

## インストール

ビルド不要でそのまま読み込めます。

1. `chrome://extensions/` を開く
2. 右上の **デベロッパーモード** を有効にする
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. このリポジトリのフォルダを選択

インストール後、ツールバーの拡張機能アイコンをクリックするとサイドパネルが開きます。

## 使い方

### ワークフローの作成・管理

- サイドパネル上部のドロップダウンでワークフローを切り替え
- **+** で新規作成、**⊕** で複製、**×** で削除
- **⇅** ボタンでワークフローの並べ替え（ドラッグ＆ドロップ）
- ワークフロー名・実行間隔（ミリ秒）・エラー時停止の設定が可能

### ステップの追加・編集

各ステップでは以下の操作タイプを選択できます。

| タイプ | 説明 |
|--------|------|
| **クリック** | 指定した要素をクリック |
| **ホバー** | 指定した要素にマウスオーバー |
| **入力** | フォーム要素に値を入力（テキスト・テキストエリア対応） |
| **コピー** | 要素のテキストを取得（リンクの場合は URL も取得） |

#### 要素の指定方法

- **Pick ボタン**: ページ上の要素を直接クリックして CSS セレクタを自動生成
- **Monitor ボタン**: DOM の変更をリアルタイム監視し、動的に追加された要素を検出
- 手動で CSS セレクタを入力することも可能

#### ステップのオプション

- **selectorIndex**: 同じセレクタに複数の要素がマッチする場合、N 番目の要素を指定（1始まり）
- **個別の実行間隔**: ステップごとに異なる待機時間を設定可能
- **複数行入力**: 入力タイプでテキストエリアモードに切り替え可能

ステップは **↑↓** ボタンやドラッグ＆ドロップで並べ替えできます。

### ワークフローの実行

1. **▶** ボタンで実行開始
2. 各ステップが順番に実行され、進捗バーとステータスアイコンで状況を確認
   - ○ 待機中 → ⏳ 実行中 → ✓ 完了 / ✗ エラー
3. **■** ボタンで途中停止
4. コピータイプのステップがある場合、実行完了後にまとめてクリップボードにコピー

#### コピー結果のフォーマット

ラベルを設定した場合:

```
■ラベル1
コピーした文字列
■ラベル2
コピーした文字列
```

ラベルを設定しない場合:

```
コピーした文字列
コピーした文字列
```

### インポート・エクスポート

- **エクスポート**: 現在のワークフローを JSON ファイルとして保存
- **インポート**: JSON ファイルからワークフローを読み込み（バリデーション付き）

## ファイル構成

```
StepControl/
├── manifest.json           # 拡張機能の設定
├── background/
│   └── service-worker.js   # ワークフロー実行の制御
├── content/
│   ├── content.js          # DOM 操作（クリック・入力・コピー）
│   ├── picker.js           # 要素ピッカー（セレクタ自動生成）
│   └── observer.js         # DOM 変更監視
├── sidepanel/
│   ├── sidepanel.html      # サイドパネル UI
│   ├── sidepanel.css       # スタイル
│   └── sidepanel.js        # UI ロジック・状態管理
├── icons/                  # 拡張機能アイコン
├── docs/
│   └── specification.md    # 仕様書
└── test/
    └── test-page.html      # テスト用ページ
```

## 既知の制約

- Service Worker の 30 秒制限により、25 秒を超える実行間隔は正常に動作しない可能性があります
- 要素の出現待ち・条件分岐・ループには対応していません
- ページ遷移後のステップ継続、iframe 内の操作、新規タブへの切替はできません
- キーボード操作（Enter / Tab / Escape 等）、ファイルアップロード、ドラッグ＆ドロップには非対応
